# ⏳ Pengingat Waktu (Time Since Counter)

Aplikasi web sederhana untuk menghitung waktu yang telah berlalu (hari, jam, menit, detik) sejak suatu momen penting secara _real-time_. Awalnya dibuat untuk menghitung waktu sejak "Seminar Proposal", namun kini dapat disesuaikan untuk momen apa saja!

## ✨ Fitur Utama

- ⏱️ **Penghitung Real-time**: Memantau hari, jam, menit, dan detik yang telah berlalu.
- ✏️ **Judul Dinamis**: Klik langsung pada teks judul untuk mengubah dan menyesuaikannya dengan momen Anda.
- 💾 **Penyimpanan Lokal**: Waktu yang sedang berjalan dan perubahan judul akan tersimpan otomatis di _browser_ (menggunakan `localStorage`), sehingga aman meski halaman di-_refresh_.
- 🌓 **Tema Gelap & Terang**: Dilengkapi dengan tombol untuk beralih antara _Dark Mode_ dan _Light Mode_.
- 📲 **Bagikan (Share)**: Bagikan progres hitungan waktu Anda dengan mudah ke media sosial atau via WhatsApp dengan Web Share API.
- 🎵 **Efek Suara Interaktif**: Dilengkapi dengan efek suara klik (`click-sound.mp3`) pada setiap tombol untuk pengalaman pengguna yang lebih baik.
